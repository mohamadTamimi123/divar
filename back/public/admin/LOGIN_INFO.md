# 🔐 اطلاعات ورود به پنل مدیریت

## 📋 اطلاعات ورود پیش‌فرض

### 👤 مدیر ارشد (Super Admin)
- **ایمیل:** `admin@divar.com`
- **رمز عبور:** `admin123`
- **نقش:** Super Admin
- **دسترسی‌ها:** کامل (تمام بخش‌ها + مدیریت کاربران)

## 🚀 نحوه دسترسی

### 1. دسترسی به صفحه ورود
```
http://localhost:5001/admin/login.html
```

### 2. وارد کردن اطلاعات
- ایمیل: `admin@divar.com`
- رمز عبور: `admin123`
- یا استفاده از دکمه "تکمیل خودکار" در صفحه ورود

### 3. دسترسی به بخش‌های مختلف
پس از ورود موفقیت‌آمیز:
- **داشبورد اصلی:** `http://localhost:5001/admin`
- **پنل کراولر:** `http://localhost:5001/admin/crawler`

## ⚙️ ایجاد کاربر مدیر (در صورت عدم وجود)

### روش 1: اسکریپت خودکار
```bash
cd back
node create-admin.js
```

### روش 2: Seeder
```bash
cd back
npx sequelize-cli db:seed:all
```

## 🔒 نکات امنیتی

### ⚠️ مهم!
- **حتماً پس از اولین ورود رمز عبور را تغییر دهید**
- از رمزهای قوی استفاده کنید
- دسترسی به پنل مدیریت را محدود کنید

### 🛡️ تغییر رمز عبور
1. وارد پنل مدیریت شوید
2. به بخش "مدیریت کاربران" بروید
3. کاربر admin را ویرایش کنید
4. رمز عبور جدید تنظیم کنید

## 🎯 سطوح دسترسی

### Super Admin
- مدیریت کامل کاربران
- مدیریت ملک‌ها
- مدیریت پرداخت‌ها
- مدیریت کراولر
- مشاهده گزارشات
- تنظیمات سیستم

### Admin
- مدیریت ملک‌ها
- مدیریت کراولر
- مشاهده گزارشات
- مدیریت محدود کاربران

### User
- مشاهده ملک‌ها
- ثبت درخواست

## 🆘 حل مشکل

### مشکل ورود نکردن
1. بررسی صحت ایمیل و رمز عبور
2. بررسی اتصال به پایگاه داده
3. اجرای اسکریپت ایجاد کاربر مدیر

### فراموشی رمز عبور
1. اجرای مجدد اسکریپت `create-admin.js`
2. یا تغییر مستقیم در پایگاه داده

### خطای دسترسی
1. بررسی نقش کاربر (باید admin یا super_admin باشد)
2. بررسی فعال بودن کاربر (isActive = true)

## 📞 پشتیبانی

در صورت مشکل در ورود یا دسترسی:
1. بررسی لاگ‌های سرور
2. اجرای اسکریپت ایجاد کاربر مدیر
3. بررسی تنظیمات پایگاه داده
